# Practice Zig

```bash
$ zig version
0.9.1

# Create Project
$ mkdir practice-zig/ && cd $_
$ zig init-exe

# Exec Source
$ zig run ./src/main.zig

# Build Then Run
$ zig build run

# Build And Run
$ zig build
$ ./zig-out/bin/practice-zig

# Format
$ zig fmt ./

# Unit Test
$ zig test ./src/main.zig
```
